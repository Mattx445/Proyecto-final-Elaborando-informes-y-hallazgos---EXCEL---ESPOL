# Proyecto Integrador – Análisis de Ventas - EXCEL

Este repositorio contiene el desarrollo del proyecto integrador del Grupo 10, enfocado en el análisis de ventas, campañas de marketing digital y la satisfacción del cliente, con el objetivo de comprender el comportamiento del consumidor y apoyar la toma de decisiones basada en datos.

## Objetivos del proyecto
- Analizar los métodos de pago digitales y su impacto en las ventas.
- Evaluar la efectividad de las campañas de marketing digital.
- Relacionar la satisfacción del cliente (ratings altos) con los picos de ventas.
- Obtener conclusiones que aporten a la mejora de estrategias comerciales.

## Contenido del repositorio
- Archivos de análisis de ventas.
- Presentación del proyecto integrador.
- Conclusiones obtenidas a partir del análisis de datos.


## Herramientas utilizadas
- Microsoft Excel
- Microsoft PowerPoint

## Conclusiones
El análisis permitió identificar patrones relevantes entre las campañas digitales, los métodos de pago y la satisfacción del cliente, evidenciando su influencia directa en el comportamiento de ventas.

## Diagrama de flujo


    A[Inicio del Proyecto] --> B[Recolección de datos de ventas]
    B --> C[Identificación de métodos de pago digitales]
    C --> D[Análisis de campañas de marketing digital]
    D --> E[Evaluación de satisfacción del cliente]
    E --> F[Comparación ratings vs picos de ventas]
    F --> G[Análisis de resultados]
    G --> H[Obtención de conclusiones]
    H --> I[Fin del Proyecto]
